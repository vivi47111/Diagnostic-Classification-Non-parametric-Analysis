# Medical Data Classification & Non-parametric Smoothing Analysis

## 1. Project Overview
This repository contains a two-part statistical study: 
1. A comparative diagnostic analysis of the **Indian Liver Patient Dataset (ILPD)**.
2. A non-parametric regression study on **Bone Mineral Density (BMD)** data focusing on the bias-variance tradeoff in curve smoothing.

## 2. Key Technical Highlights
* **Classifier Benchmarking**: Evaluated **LDA** vs. **Logistic Regression** for binary medical diagnosis, emphasizing stability under biological variance.
* **Roughness Penalty Optimization**: Implemented non-parametric smoothing techniques to balance data fidelity and structural smoothness in BMD longitudinal data.
* **Cross-Validation Strategy**: Utilized **Leave-one-subject-out CV** to ensure model robustness across different individuals.

## 3. Evaluation Metrics
* Quantified performance using **AUC (Area Under Curve)**, **Sensitivity**, and **Specificity** to provide a comprehensive diagnostic profile.
